# Networking Labs

This folder contains my practical networking labs, configurations, and hands-on exercises.

## Lab Environment

The labs are performed using:

- Cisco Packet Tracer
- Virtual Machines
- Network Simulation Tools

## Topics Covered

- Basic Network Topologies
- Router Configuration
- Switch Configuration
- VLAN Configuration
- Inter-VLAN Routing
- DHCP Configuration
- NAT Configuration
- Static Routing
- Dynamic Routing
- Network Troubleshooting

## Lab Documentation

Each lab includes:

- Network topology
- Device configuration
- IP addressing scheme
- Commands used
- Testing results
- Troubleshooting steps

## Purpose

The purpose of these labs is to develop practical networking skills and understand how real-world networks are designed, configured, and managed.
