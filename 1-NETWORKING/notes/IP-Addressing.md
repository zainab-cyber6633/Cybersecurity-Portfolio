# Introduction

An IP address ( Internet Protocol) is a unique number assigned to a device on 
a network .
 # key fact : without an  IP address , devices  cannot communicate over a network.
_____

# Types of IP address 
## IPv4 
IPv4 uses 32-bit address .
Example: 192.168.1.1

##IPv6 
IPv6 uses 128-bit address
Example: 2001:db8::1
____
# IPv4 address classes 
## Class A 
used for large networks .
Example : 10.0.0.0

## Class B 
used for medium networks 
Example: 172.16.0.0
## Class C 
used for small networks .
Example :
192.168.1.0
## Class E&D is reserved for future use.
_____
# Private IP  Addresses 
private ip address are used inside local networks.

Ranges :
- 10.0.0.0 - 10.255.255.255
- 172.16.0.0 - 172.31.255.255
- 192.168.0.0 - 192.168.255.255

____

# Public IP Address
A public IP address is designed by an Internet Service provider and is accessible over the Internet .

____
# Security Importance 
understanding ip addressing helps in :

- Firewall rules

- Security monitoring

IP addressing is a fundamental concept of networking that allows devices to communicate and exhange data.
